Description

  • Caribean Medical

Caraibean Médical, une entreprise exerçant dans la vente de produits pharmaceutiques à souhaiter innover dans la gestion de ses vente par la mise en place d’un site web capable de traiter efficacement les commandes des clients. Pour ce faire elle a contacté l’équipe d'Élite Buro pour mettre en place une application web où il sera possible d’effectuer des commandes, des analyses sur les achats, d’enregistrer les utilisateurs et leur permettre de trouver facilement les produits qu’ils recherchent. Ce projet m'a été confié.
